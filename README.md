# odin_diary
모바일 게임 오딘을 플레이하면서 하루동안 얻은 아이템들을 기록하는 일기장

<h3>프로젝트 배포 링크: https://port-0-odin-diary-dist-1jx7m2gldba9am1.gksl2.cloudtype.app/</h3>

화면 구동 결과

## backend 실행
### backend 폴더로 이동
```
cd backend
```
### app.js 파일 실행
```
node app.js
```

## frontend 실행
### frontend 폴더로 이동
```
cd frontend
```
### vue 파일 실행
```
npm run serve
```
